# PRATICA4_HTML
# 🚀 Jornada Web: Missão Marte

Este repositório contém o projeto final desenvolvido durante as aulas de **Desenvolvimento Web Estático**. O objetivo foi transformar uma notícia simples em uma Landing Page moderna, funcional e semanticamente correta.

## 📍 Evolução do Projeto
O projeto foi construído de forma incremental, passando por 4 etapas principais:
1. **Estrutura Básica:** Uso de títulos, parágrafos, imagens e listas.
2. **Dados e Coleta:** Implementação de tabelas de preços e formulários de reserva.
3. **Estilização (CSS3):** Introdução ao design, cores, fontes, box model e efeitos de interatividade (hover).
4. **Semântica e Layout:** Organização do código com tags estruturais (`header`, `main`, `section`, `footer`) e layout moderno em cards.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica e formulários.
* **CSS3:** Estilização de componentes, posicionamento (sticky), efeitos de hover e responsividade básica.

## 💡 Conceitos Aplicados
* **Semântica:** Melhores práticas para SEO e acessibilidade.
* **UX/UI:** Melhoria de contraste, legibilidade e feedback visual ao usuário.
* **Âncoras Internas:** Navegação por IDs para uma experiência de página única (Single Page).

## 🚀 Como visualizar
1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
